Identity

User Prompt Template
Default: Task ID, title, and description are injected automatically.
▸ Available template variables
{task_id} — Task's unique ID
{task_title} — Task title
{task_description} — Full description
{task_stage} — Current stage key
{task_project} — Project name
{card.plan} — Planning stage output (if available in card content)
{card.diff} — Git diff of all changes in the task's worktree
{card.<key>} — Any key from the task content JSON (e.g. {card.genre})
{content_<key>} — Alias for {card.<key>} (e.g. {content_genre})

Curly braces and brackets in injected values are automatically escaped.
